## O que deve ficar aqui

- gráficos finais de EDA;
- heatmaps;
- timelines;
- figuras comparativas de modelos;
- imagens prontas para uso em relatório ou apresentação.

## Exemplos de artefatos esperados

- `heatmap_correlacao.png`
- `timeline_ocupacao.png`
- `comparacao_modelos.png`
- `densidade_por_janela.png`

## Diretriz de uso

- os nomes dos arquivos devem ser autoexplicativos;
- versões finais devem ser priorizadas sobre imagens temporárias;
- este diretório deve servir como fonte visual para `reports/eda/`, `reports/models/` e `reports/defesa/`.
