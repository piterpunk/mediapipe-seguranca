# Estratégia de testes

Este diretório concentra os testes automatizados mínimos do projeto.

## Objetivo atual

Garantir que a pipeline base execute, gere saída e mantenha uma estrutura estável para evolução incremental.

## Evolução esperada

- testes de ingestão com amostras de vídeo;
- testes de transformação e agregação de features;
- testes de consistência de rótulos e colunas esperadas;
- testes de regressão para preservar o comportamento da pipeline.
